How to run

1. Open XAMPP and run mysql and Apache
2. You will first have to create a database named usernames and a table named users
3. For the users table, create the following variables with the following settings:
   id int auto increment
   username varchar length: 255
   email varchar length: 255
   password varchar length: 255
4. Open two terminals
5. In the first enter cd Frontend and enter npm run dev
6. In the second enter cd Backend and enter npm start
7. In the App.jsx file you will want to change the IPs to either localhost or your PC's IP address
